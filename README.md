## Анализ банковских данных

Проект представляет собой исследовательский анализ данных о банковских клиентах и их транзакционной активности с помощью
Apache Spark.

Цель — получить общую картину клиентской активности, доходов, расходов и поведения при использовании банковских услуг.

### Датасеты:

- loans.csv — информация о кредитах, доходах и возвратах чеков.
- credit_cards.csv — данные о клиентах, имеющих право на получение кредитной карты.
- transactions.csv — транзакции клиентов: снятия, пополнения, остатки на счетах.

### Технологии

- Python
- Apache Spark (Pyspark)
- pandas, NumPy
- Matplotlib, Seaborn
- Jupyter Notebook
